FE BucketList App

A simple vanilla JavaScript front end consuming and displaying todo based JSON data accessed by a RESTful Node API backend. Working across the whole stack I utilized MongoDb to store my JSON documents in a collection, then used NPM to install the Express framework into my node application to make the most of the handy routing functionality. The routes for the various CRUD operations allowed me to manipulate the todos with GET/POST/PUT & DELETE HTTP requests which all linked up to HTML inputs on my front end. GitHub | Live
